# 实验文件说明

## 文件结构

```css
file.zip
├── readme.md # 文件说明
├── main.c	# 主程序
├── similar.c	# 相似程序
├── different.c	# 差异化程序
└── keywords.txt # 关键词表
```

## 实验要求

- 使用适当的哈希函数构建哈希表, 哈希表长度为**43**。
  - *推荐使用 $(Hash(key)=key[0]*100+key[-1])\%41$, 即实验文档示例的函数*
  - `keywords.txt` 文件用作程序中的关键字的输入列表。
- 根据哈希表中关键字的频率计算 `main.c` 与其他两个程序向量几何距离和相似度。
